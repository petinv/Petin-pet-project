Добро пожаловать на мой Pet-проект!


Это веб-приложение, разработанное на Java, работающее на бесплатном сервере Oracle Cloud. Я создал собственный сервер и постарался сделать этот сайт полноценным проектом, доступным всегда.

Особенности моего проекта включают в себя возможность просмотра, добавления и изменения данных в таблице из базы данных SQL. Я хочу подчеркнуть, что не претендую на идеальное формирование кода, и этот проект создан в первую очередь для обучения и экспериментов.

Я готов рассмотреть возможности сотрудничества и попытки применения этих навыков на практике. Моя цель - попасть в команду профессионалов и обучаться вместе с опытными специалистами. Мой проект - это лишь начало моего пути, и я горячо желаю развивать свои навыки и знания, работая в структурированной и опытной команде.




Виртуальная машина:
  • Развернута виртуальная машина Ubuntu Server на Oracle Cloud.
  • Осуществлена начальная конфигурация с установкой Java и настройкой firewall.
  • Реализован проброс портов через Nginx.
  • Настроен автоматический запуск приложения в качестве службы.

База данных:
  • Настроен доступ к SQL базе данных через Java.
  • Реализован вывод данных из таблицы.
  • Добавлена функциональность управления данными: добавление, изменение, удаление строк как вручную, так и через файлы JSON.

Форма обратной связи:
  • Создана удобная форма обратной связи для отправки сообщений на мою почту.

Форма простых вычислений:
  • Добавлена форма для работы с массивами (создание, генерация, сортировка).
  • Внедрен функционал просчета стоимости скидки и доставки с учетом общей суммы.
